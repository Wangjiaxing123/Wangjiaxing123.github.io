#概论

问题求解

  • 问题抽象》数据抽象》算法抽象》 数据结构+算法，进行程序设计

    算法

  • 算法特性
    • 通用性
      • 对参数化输入进行问题求解
      • 保证计算结果的正确性
      • 有效性
      • 算法是有限条指令组成的指令序列
      • 优一系列具体步骤组成
      • 确定性
      • 算法描述中的下一步应执行的步骤必须明确
    • 有穷性
  • 基本算法分类
    • 穷举法
      • 顺序找K值
    • 回溯，搜索
      • 8皇后，树和图遍历
    • 递归分治
      • 二分找K值，快速排序，归并排序
    • 贪心法
      • huffman编码树，最短路Dijkstra算法，最小生成树Prim算法
    • 动态规划
      -最短路Floyd算法

数据结构

**结构**：实体+结构
  • 数据机构：
    -按照逻辑关系组织起来的一批数据
    • 按一定的存储方法把它存储在计算机中
    • 在这些数据上定义了一个运算的集合

数据结构的逻辑组织

  • 线性结构
    • 线性表（表，栈，队列，串等）
  • 非线性结构
    • 树（二叉树，huffman树，二叉检索树等）
    • 图（有向图，无向图）
      • 图》树》二叉树》线性表
